# 应用特性
- 支持MAC OS和Windows OS快捷键
- 支持应用快捷键，包括
  - Obsidian（Mac & Win）
  - VSCode（Mac & Win）
  - Magnet（Mac）
  - tmux (终端应用)
  - vim（终端应用）
  - welink（Win）
  - Directory Opus （Win）
  - Follow （Mac & Win）
  - Word，PPT，Excel（Mac & Win）
  - Polybox（Chrome）
  - UPDF（Mac & Win）
  - XYPlorer（Win）
  - PDF Expert（Mac）
  - JetBrains IDE (Mac & Win)
